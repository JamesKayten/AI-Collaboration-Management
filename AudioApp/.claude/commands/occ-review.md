# OCC Review Request

This command is for TCC to request OCC review and guidance.

## When to Use
- Frontend modifications needed
- Architectural decisions required
- Integration issues found
- Design changes proposed
- Test failures unclear

## Review Request Format

### Issue Summary
[Brief description of what needs review]

### Context
- What were you testing?
- What did you expect?
- What actually happened?

### Technical Details
- Error messages
- Log output
- Test results
- Code involved

### Proposed Solution (if any)
[Your recommendation]

### OCC Decision Needed
[Specific question or decision point]

## OCC Maintains Control Over
- SwiftUI code and design
- Frontend architecture
- UI/UX decisions
- ViewModel structure
- Animation and styling
- API client design

## TCC Can Handle
- Backend bug fixes
- Test improvements
- Documentation
- Backend performance
- Database optimization

---

**Remember**: OCC has final say on architecture and frontend. TCC focuses on testing and backend validation.
