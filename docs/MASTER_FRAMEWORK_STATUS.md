## What This Framework Is

A minimal file-based AI collaboration system.

**Core files:**
- `BOARD.md` - Current status
- `TASKS.md` - Task list

**Total code:** 185 lines

---

## How to Use

### Check Status
```bash
/check-the-board
# or
cat BOARD.md TASKS.md
```

### Update Status
```bash
edit BOARD.md
edit TASKS.md
git commit -m "Update status"
git push
```

**That's it.**

---

## Available Commands

- `/check-the-board` - Read status and tasks
- `/fix-violations` - Fix reported issues
- `/verify` - Check fixes work
- `/works-ready` - Validate and merge
- `/merge-to-main` - Create PR

All commands are simple file operations. No scripts required.

## Framework Philosophy

1. **KISS** - Keep It Simple, Stupid
2. **Files over scripts** - Read markdown, not JSON
3. **Trust over verify** - If it works, it's done
4. **Clarity over automation** - Simple beats clever
5. **Minimal over comprehensive** - Less is more
